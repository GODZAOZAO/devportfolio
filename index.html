Distributed TF-IDF Document Search

This system utilizes the TF-IDF algorithm to construct a distributed document search system, delivering highly efficient relevance-based search capabilities. To ensure fault tolerance and reliability, we integrated Apache ZooKeeper's SEQUENCE|EPHEMERAL flags to implement a leader election algorithm, effectively preventing node disconnection or failure. This project demonstrates our commitment to creating cutting-edge solutions that optimize search efficiency while maintaining system resilience in distributed environments.
